## Why you should contribute ?

Dengan berkontribusi, Anda dapat membantu membuat materi yang lebih bermanfaat bagi mahasiswa Teknik Informatika dan membangun portofolio yang kuat. Serta, ini adalah kesempatan untuk belajar dari praktik terbaik, berkolaborasi dengan orang lain, dan memperluas jaringan Anda dalam dunia pengembangan. Mari bergabung dan menjadi bagian dari upaya ini untuk meningkatkan pengalaman pembelajaran teknik informatika bagi semua orang!

## How to contribute ?

Adapun syarat yang harus dipenuhi untuk berkontribusi dalam proyek ini adalah sebagai berikut:

- Ada akun GitHub.
- Memiliki pengetahuan dasar tentang Git dan GitHub.
- Anda harus memiliki pengetahuan dasar tentang code [Markdown](https://www.markdownguide.org/cheat-sheet/).
- Pengetahuan dasar tentang [Node.js](https://nodejs.org/en), [Next.js](https://nextjs.org/docs),dan [Nextra](https://nextra.site/docs).
- Terinstal global `pnpm package` di mesin lokal anda. ([Cek disini](https://pnpm.io/installation) untuk instalasi pnpm package).
- Menerapkan konsep `clean code` dan `best practice` dalam pengembangan open source ini.

Anda dapat berkontribusi dengan cara berikut:

1. Fork repository ini.
2. Clone repository yang telah di fork ke mesin lokal anda.
3. Buat branch baru dengan nama yang sesuai dengan kontribusi anda.
4. Lakukan perubahan yang diperlukan.
