# What is Informatics Resources?

Informatics rosurces ditujukan bagi mahasiswa teknik informatika untuk membantu menemukan sumber bacaan dalam menjalani perkuliahan. Informatics resources ini berisi materi serta referensi tugas yang berkaitan dengan teknik informatika, seperti pemrograman, jaringan, dan lain-lain.

## How to Contribute

1. Fork repositori dan clone ke mesin lokal Anda.
2. Buatlah branch baru untuk fitur atau perbaikan bug yang akan Anda kerjakan.
3. Lakukan perubahan dan pastikan anda menerapkan `clean code` dan `best practice`.
4. Buat pull request.
